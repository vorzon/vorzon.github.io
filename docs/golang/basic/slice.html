# 简单介绍

slice 表示一个拥有相同类型元素的可变长度的序列

slice 三个重要属性：指针、长度、容量

# 常用操作

# 初始化

// make初始化，元素会初始化为T类型零值
s := make([]T, len)
s := make([]T, len, cap)

// 字面量初始化，和数组的区别是不指定长度
s := []T{}

// 从原有slice截取
s1 := s[i:j]

// copy函数

# slice 结构

type slice struct {
	array unsafe.Pointer    // 实际数据，底层是数组
	len   int               // slice实际元素个数
	cap   int               // slice容量，len <= cap，扩容其实就是扩cap
}

# 扩容

// cap 期望容量
func growslice(et *_type, old slice, cap int) slice {
    ......

    newcap := old.cap
    doublecap := newcap + newcap
    // 如果期望容量大于slice原来容量cap的2倍，则使用期望容量
	if cap > doublecap {
		newcap = cap
	} else {
        // 如果原来容量小于1024(1M)，则扩容2倍
		if old.cap < 1024 {
			newcap = doublecap
		} else {
			// 如果大于1024，则循环每次增加原来容量的1/4，直到大于期望容量
			for 0 < newcap && newcap < cap {
				newcap += newcap / 4
			}

			if newcap <= 0 {
				newcap = cap
			}
		}
    }

    ......
}

总结：

  1. 如果期望容量大于 slice 原来容量 cap 的 2 倍，则使用期望容量
  2. 否则，如果原来容量小于 1024(1M)，则扩容 2 倍
  3. 如果大于 1024，则循环每次增加原来容量的 1/4，直到大于期望容量
